ZimDroid

An Android client for the Zim desktop wiki application.

Author: Frank Van Geirt <frank.van.geirt@firewolf.be>

Project page: https://launchpad.net/zimdroid

INSTALLATION
- Zim notebook on desktop on Dropbox
- On Android, the Dropbox app and the Dropsync app needs to be installed
- Start up Dropsync and create a local folder
- Link this local folder to the Zim folder in Dropbox
